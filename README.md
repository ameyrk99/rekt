### Dependencies:
- [Qt](https://www.qt.io)
- [Material Design Icons Font](https://materialdesignicons.com)

### Compile

```
qmake Books.pro -spec linux-g++ CONFIG+=qtquickcompiler
make
```

### Run
```
./Recommui
```

### Make Clean:
```
make clean
```
